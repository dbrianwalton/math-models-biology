Example 2.5.9.

Use computer analysis to find the fixed points for the discrete logistic model \(P_{n+1} = (1+r_0)P_n - a P_n^2\text{.}\) Find the slopes \(f'(x^*)\) for each fixed point and then discuss how the stability depends on the values of the parameters.
Solution.
The projection function is \(f(x) = (1+r_0) x - a x^2\text{,}\) which is a function of \(x\) involving two additional parameters \(r_0\) and \(a\text{.}\) To perform our analysis using Sage, we will need to declare all three symbols as mathematical variables. Nevertheless, we can still solve the fixed point equation for \(x\text{,}\) and Sage will provide answers in terms of the paramters.
When you evaluate that work, we will match results we would get using basic calculus. If \(f(x) = (1+r_0)x - ax^2\text{,}\) we know how to find a derivative as
\begin{equation*} f'(x) = (1+r_0) - 2 a x\text{.} \end{equation*}
Similarly, we can algebraically write down the fixed point equation and solve it by hand:
\begin{align*} f(x) \amp = x \\ (1+r_0)x - a x^2 \amp = x \\ x+r_0 x - a x^2 \amp = x \\ r_0 x - a x^2 \amp = 0\\ x(r_0 - a x) \amp = 0 \end{align*}
Because this equation says the product of \(x\) with \(r_0 - ax\) equals zero, the solutions are found by setting each factor individually to zero, giving us \(x=0\) and \(x = \frac{r_0}{a}\) as our two solutions. This matches what Sage tells us.
From our Sage work, or by evaluating \(f'(0)\text{,}\) we find \(f'(0) = 1+r_0\text{.}\) This means that whenever \(r_0 \gt 0\text{,}\) \(f'(0) \gt 1\text{.}\) By TheoremÂ 2.5.8, we now know that \(x^* = 0\) is an unstable fixed point whenever \(r_0 \gt 0\) It is actually possible to have \(-1 \lt r_0 \lt 0\) if the per capita death rate exceeds the per capita birth rate. In that circumstance, we have \(0 \lt f'(0) \lt 1\text{.}\) Consequently, \(x^*=0\) is a stable fixed when the death rate exceeds the birth rate for \(P=0\text{.}\)
For the second fixed point \(x^* = \frac{r_0}{a}\text{,}\) we have \(f'(x^*) = 1 - r_0\text{.}\) We think through the same conditions as above. If \(-1 \lt r_0 \lt 0\) (negative \(r_0\)), we have \(f'(x^*) \gt 1\) but we also have a fixed point \(x^* \lt 0\) with a negative value. In the current setting for populations, \(P\lt 0\) does not make sense. But mathematically, such a negative fixed point would be unstable. If \(0 \lt r_0 \lt 1\text{,}\) we have \(f'(x^*)\) between 0 and 1, which means the fixed point is locally stable and sequences converge without oscillation. Once we have \(r_0 \gt 1\text{,}\) we have \(f'(x*) \lt 0\) so that nearby sequences oscillate above and then below the fixed point. For \(1 \lt r_0 \lt 2\text{,}\) we have \(|f'(x^*)| \lt 1\) and the oscillation decays away to the stable fixed point. Once we have \(r_0 \gt 2\text{,}\) this causes \(|f'(x^*)| \gt 1\) such that oscillation grows and the fixed point is locally unstable.
