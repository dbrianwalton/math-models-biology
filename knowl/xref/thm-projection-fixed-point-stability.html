Theorem 2.5.8. Stability of Fixed Points for Projection Functions.

Suppose a projection function \(f\) has a fixed point at \(x=x^*\) and \(f'(x^*)\) exists. So long as \(|f'(x^*)| \ne 1\text{,}\) the local stability of the fixed point is determined by the value of \(f'(x^*)\) as follows.
  • If \(|f'(x^*)| \lt 1\text{,}\) then \(x^*\) is a locally stable fixed point.
  • If \(|f'(x^*)| \gt 1\text{,}\) then \(x^*\) is a locally unstable fixed point.
